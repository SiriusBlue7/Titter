Para abrir las bases de datos se usa el comando:

mysql -h mysql.lab.it.uc3m.es -u 21_comweb_11 -p -D 21_comweb_11b

Con la contrraseña:

1LU0KI1i

Las distintas bases de datos van desde 

21_comweb_11a ----> Base de datos usada para los ejemplos de clase

21_comweb_11b ----> Base de datos usada para hacer los ejercicios de clase

21_comweb_11c ----> Base de datos usada para el ejercicio 4s

21_comweb_11d ----> Base de datos usada para el PROYECTO

Para ver todas las tablas que tiene una base de datos usamos :

SHOW TABLES

para ver todos los elementos que tiene una tabla:

SELECT * FROM nombre_de_tabla

Para añadir archivos .sql
mysql -h mysql.lab.it.uc3m.es -u 21_comweb_11 -p -D 21_comweb_11c <tablas.sql

mysql -h mysql.lab.it.uc3m.es -u 21_comweb_11 -p -D 21_comweb_11c <datos.sql
